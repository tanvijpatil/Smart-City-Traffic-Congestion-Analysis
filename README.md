# Smart City Traffic Congestion Analysis

## Overview
This project analyzes urban traffic data using Python to identify congestion patterns, peak traffic hours, and vehicle distribution. The goal is to derive insights that can help improve traffic management in smart cities.

---

## Tech Stack
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## Features
- Traffic analysis based on hour, day, and date  
- Visualization of vehicle type distribution  
- Heatmap to identify peak congestion periods  
- Analysis of traffic situations (Low, Medium, High)  

---

## Key Insights
- Peak traffic observed during morning and evening hours  
- Cars contribute the highest traffic volume  
- Weekdays experience higher congestion than weekends  
- Traffic significantly reduces during late night hours  

---

## Applications
- Helps in traffic signal optimization  
- Supports urban planning decisions  
- Can reduce traffic congestion and travel time
